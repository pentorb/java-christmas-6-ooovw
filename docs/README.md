# 12월 이벤트 플래너

-----
우테코 식당의 2023년 12월 이벤트를 위한 이벤트 플래너



## 프로젝트 개요

-----
- 예상 식당 방문날짜를 입력합니다.
- 주문할 메뉴와 개수를 입력합니다.
- 주문 메뉴, 할인 전 총주문 금액, 증정 메뉴, 혜택 내역, 총혜택 금액, 할인 후 예상 결제 금액, 12월 이벤트 배지를 출력합니다.


## 기능 목록

-----
### 날짜 입력 기능
- [x] 사용자로부터 날짜를 입력 받는 기능
  - [x] 숫자가 아닌 값이 입력되는 경우를 검증
  - [x] 1보다 작고 31보다 큰 숫자가 입력되는 경우를 검증

### 메뉴 입력 기능
- [x] 사용자로부터 메뉴와 개수를 입력 받는 기능
  - [x] 없는 메뉴가 입력되는 경우를 검증
  - [x] 메뉴의 개수가 1보다 작은 숫자가 입력되는 경우를 검증
  - [x] 예시와 다른 형식으로 메뉴가 입력되는 경우를 검증
  - [x] 메뉴를 중복하여 입력하는 경우를 검증
  - [x] 한번에 20개가 넘는 메뉴를 주문하는 경우를 검증
  - [x] 음료만 입력하는 경우를 검증

### 이벤트 플래너 출력 기능
- [x] 주문 메뉴를 수량과 함께 표시하는 기능
- [x] 주문 메뉴의 가격을 더해서 총 주문 금액을 구하는 기능
- [x] 할인 전 총주문 금액이 12만원 이상이면 샴페인 1개 증정하는 기능
- [x] 혜택을 내역별로 금액과 표시하는 기능
  - [x] 1,000원으로 시작하여 크리스마스가 다가올수록 날마다 할인 금액이 100원씩 추가 할인하는 기능
  - [x] 평일에는 디저트 메뉴를 메뉴 1개당 2,023원 할인하는 기능
  - [x] 주말에는 메인 메뉴를 메뉴 1개당 2,023원 할인하는 기능
  - [x] 이벤트 달력에 별이 있으면 총주문 금액에서 1,000원 할인하는 기능
  - [x] 모든 혜택의 금액을 더하여 총혜택 금액으로 표시하는 기능
- [x] 적용된 이벤트가 하나도 없다면 혜택 내역 "없음"으로 표시하는 기능
- [x] 총혜택 금액에 따라 다른 이벤트 배지를 부여하는 기능
